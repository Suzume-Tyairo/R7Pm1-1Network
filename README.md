設問2(2)
表2中の下線⑤について，R10のインタフェースに設定するOSPFの
コストをどのような値に変更すべきか，30字以内で答えよ。

実験結果
ルータR10の広域イーサ網側インタフェースにOSPFのコスト1001以上の値を設定すれば
PCからSaaSへの通信はPC → L2SW11 → L3SW10 → R11 → R31 → L2SW30 → SaaSになる。
SaaSからPCへの通信についても
ルータR10のR11側インタフェースにOSPFのコスト911以上
ルータR10のL3SW10側インタフェースに921以上の値を設定すれば
SaaS → L2SW30 → R30 → R31 → R11 → L3SW10 → L2SW11 → PCになる。
